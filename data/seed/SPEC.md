# 文明星图 · 知识库扩充规格(承包方必读)

你在为「文明星图」(人类迁徙 × 七大领域的全球史交互图)批量产出**结构化知识条目**。
产物 = 一个纯 JSON 文件,落在 `B:\AI项目\12_文明史交互网络图\data\seed\` 下指定文件名。
写完必须自检:`node B:\AI项目\12_文明史交互网络图\tools\validate_seed.js <你的文件>` 直到 **0 ERROR**(WARN 也要尽量消掉)。

## 1. 文件格式

节点批次(`nodes_<领域>.json`):

```json
{ "nodes": [
  { "id": "sci_optics_alhazen", "t": "光学 · 海什木", "d": "sci", "y": 1021,
    "lat": 30.0, "lon": 31.2, "w": 2,
    "gist": "用实验检验假说,《光学书》比伽利略早六百年。",
    "detail": "海什木在开罗系统研究光的反射折射,坚持以受控实验裁决理论,被视为实验方法的先驱,经拉丁译本影响了培根与开普勒。",
    "links": ["scirev"], "src": "海什木" }
] }
```

迁徙/流动批次(`arcs_extra.json`):

```json
{ "arcs": [
  { "id": "arc_austronesian", "t": "南岛语族扩散", "d": "ren", "y": -3000,
    "from": [23.7, 121.0], "to": [-17.5, -149.6], "dur": 3500, "w": 2,
    "gist": "从台湾出发的独木舟,把一个语系撒满半个地球的海洋。",
    "src": "南岛语系" }
] }
```

- 编码 UTF-8(无 BOM),纯 JSON,不要注释、不要多余字段。
- 节点字段全必填;弧的 `detail`/`links` 可选,其余必填。

## 2. 字段规则

| 字段 | 规则 |
|---|---|
| id | 节点 = `<领域>_小写slug`(如 `pol_ashoka`);弧 = `arc_小写slug`。全库唯一,禁止与《现有条目清单》重复 |
| t | 中文标题 ≤24 字。人物/文明事件用「事物 · 人名」式(如 `活字印刷 · 毕昇`)可选 |
| d | `ren人群迁徙 / eco经济 / pol政治 / phi哲学 / art艺术 / sci科学 / rel宗教神学` 之一;节点必须与文件名领域一致 |
| y | 整数年份,**负数=公元前**(前3000年 = `-3000`)。取学界通行年份;时段事件取起点或代表年 |
| lat/lon | 事件发生/起源地,1-2 位小数,必须落在真实相关位置(城市/地区),别全堆在首都 |
| w | 重要度 1/2/3。**w=3 是"改写人类轨迹"级,全批 ≤15%**;w=2 约三成;其余 w=1 |
| gist | 8~48 字。一句直白深刻:先给结论,带一点余味/反差。**禁止教科书腔、禁止空话** |
| detail | 40~170 字。信息密度优先:是什么 + 为什么改变了世界 + 一个具体细节。不复述 gist |
| links | 0~5 个 id,含义 = **它影响了谁**(指向更晚的条目)。只能指向:①《现有条目清单》里的 id;②你本批文件内的 id。跨批次关系不要写,后续统一编织 |
| src | 中文维基百科**准确条目名**(用于生成外链),≤40 字 |
| from/to | 弧线起讫 `[lat, lon]` |
| dur | 弧线持续年数(整数,5~3000),迁徙从 y 年起持续 dur 年 |

## 3. 文风(对齐现有库,三条金标准)

- `走出非洲`:gist「一个物种离开非洲,一部全球史就此起步。」
- `农业革命`:gist「学会种地,人类第一次能囤积盈余——城市、阶级、文字的总开关。」
- `古腾堡印刷术`:gist「金属活字让思想以病毒速度传播——宗教改革与科学革命的助燃剂。」

要求:中文、克制、洞察感;不用感叹号堆情绪;不出现"著名的""伟大的"这类空修饰;不提"维基/百科"。

## 4. 内容红线

1. **族群一律用「人群/迁徙/文化」框架**,不用生物学"人种"叙事;涉及争议(领土/宗教/殖民)保持史学中立,写事实与影响,不下价值判断。
2. **不编造**:年份、地点、因果都要是学界主流说法;拿不准的宁可不收。
3. 全球视野配额(校验器会查):每批 ≥4 大区(美洲/欧洲/非洲中东/南亚中亚/东亚太平洋);公元 1000 年前 ≥25%;1800 年后 ≤45%。东亚条目占 15% 左右(受众中文)。
4. 与《现有条目清单》**同一事件不要重复收录**(校验器按标题查重,换个说法也算重复,主题相同就跳过)。
5. 宁缺毋滥:凑不够数就少交,别灌水。
