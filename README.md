# 文明史交互网络图(Civilization Weave)

把「人类迁徙」与七大领域(**人群/祖源、经济、政治、哲学、艺术、科学、宗教/神学**)从史前到 2026 的发展变迁与相互影响,做成**覆盖全球地图的可交互网络图**:地图底图 + 迁徙弧线 + 知识网络节点 + 时间轴,节点可点开详情,丝滑进出场动画。中文界面。三大目标:**美观 · 知识密度 · 可互动** 三者兼顾。

> 起于 2026-07-12。数据在 B 盘,长期落地目录 = 本目录 `B:\AI项目\12_文明史交互网络图\`。

## 设计基调(定稿)

**「黑丝绒文明星图」**——深空蓝黑的地球之上,文明是光,迁徙是流动的金线,知识是星座间的细丝;界面谦让,只有数据发光。

- **配色**:墨渊底 `#0A0F1A` / 文明金(迁徙弧·高亮)/ 星链青(知识连线)/ 烽火红(冲突)/ 月白正文;每色 3 档亮度=重要度。
- **字体**:标题思源宋、UI 思源黑、拉丁与数字 Inter;引文限霞鹜文楷。
- **三铁律**:①光即信息(只有数据发光,UI/文字永不发光);②三层披露(hover 3 秒 → 侧栏 30 秒 → 外链深读,每层首句先给结论);③底图永远退后、单一隐喻到底(事件=点、流动=弧、关联=丝)。

## 技术选型(v2 已落地)

- **单文件自包含 HTML**(web/index.html,~3.2MB):globe.gl 3D 发光地球(vendor 内联,零外部请求)+ 双 2D 画布(星空底 / 节点前景)。可直接双击打开,可挂静态站。
- **架构分层**:`#scene` 星空 → `#globe` 三维(球体自发光纹理 / 历史疆界多边形 / 迁徙弧流光 / 选中脉冲)→ `#fg` 前景(节点光点 / 标签避让 / 影响线,永不截获指针)。节点每帧由 `GlobeView.syncScreen` 把球面投影写回 2D,地图↔网络两种世界共用同一套节点绘制与 morph。
- **踩过的坑(勿回退)**:①globe.gl 运行时优先绑定 `window.THREE`,外置 three 版本不符会炸(现方案零跨实例,不需要 three);②疆界 GeoJSON 是 RFC7946 逆时针外环,three-globe 要顺时针,反了会填成"全球补集"(灰幕+百万三角形+射线检测卡死),管线已强制翻转;③小邦/无名地带必须烘进球面纹理而非多边形层(巨型多边形拖垮逐帧 raycast);④纹理要标 `colorSpace='srgb'`、默认灯要在 tick 里补刀灭掉(init 时灯还没挂进场景);⑤**新版 three 的 `texture.clone()` 与原纹理共享 `source`**——给克隆体换 `image` 会连原贴图一起换掉(位移贴图接线时球面整个变成灰度高程图、西藏白斑事故),要独立纹理必须 `new (m.map.constructor)(img)`。
- **3D 地形**:`tools/build_terrain.js` 从 earth-topology 高程烘两张图——hillshade+hypso 灰度叠进陆地纹理(soft-light),纯高程 JPEG 做球体顶点位移(球面重建为 512×256 分段,`displacementScale=2.2`=半径 2.2%≈27 倍夸张;疆界层抬到 0.028、节点投影 0.03 防穿山)。想更夸张只调 displacementScale 一个数。
- **后续可选**:知识网络大规模化 d3-force→sigma.js→cosmos.gl;动效 GSAP + Motion;海量知识 sql.js-httpvfs 懒加载 SQLite。

## 关键约定(避坑)

1. **范围先纵切后横铺**:先把引擎 + 一条完整迁徙史诗(贯穿七领域)做成样板质量,再横向扩。不一上来铺全。
2. **"人种"→"人群/祖源/迁徙"框架**:现代遗传学否定生物学人种;这样表达更科学、避争议,也少触发安全网误判。
3. **每条关系标来源 + 可信度**:历史疆界与"谁影响谁"有争议,不假装精确。
4. **长期活着**:做静态站挂 GitHub Pages(peipei707,同 nekoshark),零维护;别学 Histography 域名一过期就死。
5. **桌面优先、移动降级**。
